Este repositorio contiene un pequeño proyecto web que desarrollé como mi propio “link‑in‑bio” o página de enlaces personalizada para presentar mi perfil profesional y mis redes. Es básicamente una página HTML estática muy ligera, acompañada de estilos CSS y un poco de JavaScript para alguna funcionalidad simple.

En concreto, la página muestra:

Mi nombre (“Juan Jesus”) y una breve descripción como diseñador web, enfocada en negocios que quieren hacerse notar en Internet.
Algunos enlaces directos a mis perfiles y trabajos: mi LinkedIn, mi Instagram y mi portfolio personal.
Un diseño claro, minimalista y eficaz pensado para que cualquier visitante que llegue desde un enlace en redes sociales pueda acceder fácilmente a mis perfiles y trabajos sin distracciones.

Esta estructura (con index.html, los estilos y un app.js) deja claro que el objetivo es ofrecer esta página de presentación/puerta de entrada a mis contactos digitales.

En resumen: es una landing page personal de enlaces que uso para centralizar mi presencia online y dirigir a quien me busca hacia mis perfiles profesionales y mis trabajos más relevantes.
